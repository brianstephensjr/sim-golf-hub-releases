# Sim Golf Hub — Releases

This is the public download page for **Sim Golf Hub**, a Windows desktop app that lets a phone or
tablet act as a controller for golf-simulator software on your PC.

## Download

Get the latest build from the [**Releases**](../../releases/latest) page:

1. Download `sghub-agent-<version>.zip` from the newest release.
2. Unzip it anywhere and run `sghub-agent.exe`.
3. See `SETUP.txt` inside the zip for first-run notes (Windows SmartScreen, firewall).

Sim Golf Hub checks this page on startup and lets you know in-app when a newer version is available.

## Updating

Download the newest zip, quit the running app from its tray icon, unzip the new version over the old
folder, and relaunch. Your settings are kept.

---

This repository hosts release builds only. The source code lives in a separate private repository.
